Time Range Inputs

Using a time range input is a common way to control the time bounds of a search. This example shows you can can easily set up a time range input, without any custom code. This example uses Django template tags, but the same can easily be achieved in JavaScript.

Multiple time range inputs can exist simultanenously on the page, controlling different searches (or even the same search).
