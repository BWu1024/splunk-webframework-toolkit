Custom Table Rendering

When rendering a table, it can be very powerful to be able to customize how a particular cell is rendered. For example, you can render a particular cell as a set of icons to indicate severity, or even color an entire row.

Using the BaseCellRenderer class, we can implement our own cell renderers that can be reused and shared. In this example, depending on the severity of each row, we put a custom icon to give a clear visual indication. You can also see the original table output (tied to the same search manager).
